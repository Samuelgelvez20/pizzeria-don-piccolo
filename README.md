# 🍕 Pizzería Don Piccolo

Proyecto desarrollado en MySQL para gestionar el proceso de venta y entrega de pizzas.

## Objetivos

- Gestión de clientes
- Gestión de pizzas e ingredientes
- Gestión de pedidos
- Gestión de pagos
- Gestión de domicilios
- Gestión de repartidores

## Tecnologías

- MySQL
- MySQL Workbench
- VS Code
- Git
- GitHub
